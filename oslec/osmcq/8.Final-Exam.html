Operating Systems Final Exam

Comprehensive Final Exam on OS Concepts
1
Which computer system component controls and coordinates the use of hardware among various applications and users?
Explanation
The Operating system "controls and coordinates the use of the hardware among the various application programs for the various users."
Source: Page 2, Section 1.1
2
The "stored-program computer concept," where instruction data and program data are stored in the same memory, is the basis for which architecture?
Explanation
"Von Neumann architecture is based on the stored-program computer concept, where instruction data and program data are stored in the same memory."
Source: Page 3, Section 1.2
3
What is the primary reason an operating system needs scheduling capabilities?
Explanation
When many programs run in parallel, "there may be competing and conflicting requests to access hardware resources." The OS streamlines these via scheduling.
Source: Page 4, Section 2.1
4
What is the difference between a "program" and a "process"?
Explanation
A program is "static and embedded in the form of processed code," while a "process refer to a program in execution."
Source: Page 5, Section 2.2
5
What is the common solution for the problem where a program's physical memory address can change during runtime?
Explanation
"To solve this problem, the common solution is to map the program's compiled addresses to the actual address in physical memory."
Source: Page 7, Section 3.2
6
The OS presents a unified I/O system to applications. How does it handle the wide variety of devices?
Explanation
"The OS handles this by establishing protocols and interfaces with each I/O controller."
Source: Page 8, Section 4.1
7
A software component in kernel mode that handles all interfaces with a specific device is called a:
Explanation
"Typically, there is a software component in kernel mode called as the “device driver” that handles all interfaces with a device."
Source: Page 11, Section 4.2
8
Which two main components facilitate OS file management?
Explanation
"OS file systems have two main components to facilitate file management: Directory Service... [and] Storage Service."
Source: Page 13, Section 5.1
9
A structure containing file attributes like location, size, and access controls is called the:
Explanation
"Some of these are part of the so-called file control block, which a user has access to via the OS."
Source: Page 16, Section 5.2
10
What is the primary purpose of separating the OS into different "rings" of privilege?
Explanation
"...it ensures different privilege levels are granted to programs, based on which mode they run in. OS divides the program execution privileges into different rings."
Source: Page 18, Section 6.2
11
In the protection rings model, which ring has the least privilege and is where applications typically run?
Explanation
"For example, applications and user-mode services running in Ring 3 would not be able to access the hardware directly." The diagram also shows Ring 3 as "Least Privileged".
Source: Pages 18-19, Section 6.2
12
What is the component provided by a host OS that manages hardware access for guest virtual machines?
Explanation
"The host OS provides a hypervisor, which manages the access to the hardware."
Source: Page 21, Section 7.1
13
Which of the following is NOT listed as an advantage of virtualization?
Explanation
The list of advantages on page 22 includes running legacy systems, easier machine creation, and emulation. It does not claim to increase raw performance over the physical hardware it's running on.
Source: Page 22, Section 7.1
14
The third type of memory requirement for a process arises from runtime requests from the:
Explanation
"The third type of memory requirement could arise from runtime requests for memory. These could be needed from the stack/heap for the program to perform its execution."
Source: Page 6, Section 3.1
15
A device that communicates by sending single characters (bytes or octets) is called a:
Explanation
The two broad categories are block and character devices. Character devices are defined by communication via single characters.
Source: Page 12, Section 4.3
