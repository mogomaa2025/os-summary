Section 1.3.2.1
1
How many main categories of OS based on usage type are listed in the text?
Explanation
"Based on this, there are five main categories: 1. Batch, 2. Time Sharing, 3. Parallel, 4. Network, 5. Real Time."
Source: Page 20, Section 1.3.2.1
2
Which OS category is for usages where a sequence of steps is executed repeatedly without human intervention?
Explanation
"Batch: For usages where a sequence of steps needs to be executed repeatedly without any human intervention."
Source: Page 20, Section 1.3.2.1
3
Windows, Mac, and Linux are given as examples of which OS category?
Explanation
"The OSs in such cases are categorized as time-sharing OSs. (Windows, Mac, and Linux)"
Source: Page 20, Section 1.3.2.1
4
Which OS category is needed for systems that require fine-grained time precision and responsiveness?
Explanation
"Real Time: In some cases, we need fine-grained time precision in execution and responsiveness. We call these systems real-time OSs."
Source: Page 20, Section 1.3.2.1
5
What is the key difference between Parallel and Network OS categories mentioned in the text?
Explanation
The text describes Parallel systems as "over tightly coupled systems" and Network systems as "loosly coupled".
Source: Page 20, Section 1.3.2.1
6
AIX for IBM RS/6000 and Solaris for workstations are examples of what type of OS?
Explanation
"Parallel – Distributed (over tightly coupled systems): ... (AIX for IBM RS/6000 and Solaris for workstations)"
Source: Page 20, Section 1.3.2.1
7
True or False: DOS is an example of a Time-Sharing OS. TRUE/FALSE
Explanation
This is false. The text lists DOS as an example of a Batch OS: "These classes are called batch OSs. (Old Mainframe and DOS)".
Source: Page 20, Section 1.3.2.1
8
Which OS category best describes a system where many users or applications need to share limited hardware resources?
Explanation
"Time Sharing: For systems where many users (or many applications) access common hardware, there could be a need to timeshare the limited resources."
Source: Page 20, Section 1.3.2.1
