Which protocol is defined as providing "reliable data transfer between hosts"?
Explanation
FTP (File Transfer Protocol) is described as "a transmission protocol that provides reliable data transfer between hosts".

Which protocol is described as a "Microsoft proprietary protocol" for remote connections?
Explanation
Remote Desktop Protocol (RDP) is identified as "a Microsoft proprietary protocol that enables remote connections".
Source: Page 39
3
Which mail protocol is specifically used to *send* messages to other mail servers?
Explanation
SMTP (send mail transfer Protocol) is used to "send messages back and forth to other Mail Servers or Email Clients". POP3 and IMAP4 are for retrieving mail.
Source: Page 41
4
According to the comparison table, which protocol runs on port 23 and is considered "Not Secure"?
Explanation
The "Telnet VS SSH" table clearly shows that Telnet runs on port 23 and is a "Not Secure Protocol".
Source: Page 39
5
What is a key feature of POP3 (Post Office Protocol version 3) that differentiates it from IMAP4?
Explanation
POP3 is noted to "Download messages on the hard disk" and "can work Offline". In contrast, IMAP4 is described as retrieving only the message header.
Source: Page 41
6
The primary purpose of IMAP4 (Internet Message Access Protocol version 4) is to...
Explanation
The slide explicitly states that IMAP4 is used to "Retrieve only message header".
Source: Page 41

False. Microsoft Outlook is listed as an example of a "Non web based" client. Hotmail and gmail are given as web-based examples.
