# NW and OS Summary

A simple **GitHub Pages** site that provides concise notes and quick revision material for Operating Systems (OS) and Computer Networks (NW), built entirely with static HTML. [page:1]

## Live site

- GitHub Pages: https://mogomaa2025.github.io/os-summary/ [page:1]

## Features

- Short OS summary page (`os.html`) focused on key theoretical concepts and exam-style points. [page:1]  
- Networking summary page (`nw.html`) covering fundamental network topics. [page:1]  
- Coverage pages in `oslec/` and `nwlec/` for OS and NW MCQ-style revision. [page:1]  
- Quiz answers page (`quiz-ans.html`) for quick self-check and revision. [page:1]  
- Pure HTML implementation, easy to host on any static server or GitHub Pages. [page:1][web:6]

## Project structure

- `index.html` – Landing page linking to OS and NW summaries. [page:1]  
- `os.html` – Operating systems summary content. [page:1]  
- `nw.html` – Networking summary content. [page:1]  
- `cppstrings.html` – Extra C++ strings notes as supporting material. [page:1]  
- `quiz-ans.html` – Answers for quiz questions. [page:1]  
- `oslec/` – OS lecture/coverage HTML pages (e.g., `Coverage-OS-Mcq.html`). [page:1]  
- `nwlec/` – NW lecture/coverage HTML pages (e.g., `Coverage-Mcq.html`). [page:1]

## Getting started

To run the project locally:

1. Clone the repository:
git clone https://github.com/mogomaa2025/os-summary.git
cd os-summary
2. Open `index.html` directly in your browser. [page:1]

Or just visit the live GitHub Pages site:  
https://mogomaa2025.github.io/os-summary/ [page:1]
